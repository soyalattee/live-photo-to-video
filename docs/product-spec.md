# Live Photo Short Video Creator

## iOS Swift 기능명세서 v1.0

---

## 1. 문서 목적

본 문서는 iPhone 사용자가 앨범 내 **사진 및 Live Photo**를 선택하면, 이를 자동으로 짧은 영상으로 생성하고 저장할 수 있는 iOS 앱의 MVP 기능 요구사항을 정의한다.

### 목적

- 개발 범위 명확화
- 화면별 기능 정의
- 데이터/상태 정의
- 예외 처리 기준 정의
- Swift 기반 구현 시 필요한 모듈 책임 범위 정의

---

## 2. 제품 한 줄 정의

> 사용자가 아이폰 앨범에서 여러 장의 사진과 Live Photo를 선택하면, 자동으로 쇼츠 영상을 생성해주는 앱

---

## 3. 대상 사용자

- 10~30대 iPhone 사용자
- 여행 / 카페 / 일상 기록을 자주 남기는 사용자
- SNS용 짧은 영상을 빠르게 만들고 싶은 사용자
- 영상 편집이 귀찮은 사용자

---

## 4. MVP 목표

### 핵심 목표

- 3초 내 영상 생성 경험 제공
- 편집 없이 콘텐츠 완성 경험 제공
- Live Photo를 자동으로 활용

### 성공 기준

- 생성 완료율 ≥ 70%
- 저장 전환율 ≥ 40%
- 평균 생성 시간 ≤ 5초

---

## 5. 지원 플랫폼

- iOS (Swift)
- 최소 버전: iOS 16+

---

## 6. 기능 범위

### 6.1 포함 기능 (MVP)

#### A. 미디어 선택

- 사진 / Live Photo 선택 가능
- 최소 3장 / 최대 30장
- 선택 순서 유지

#### B. 자동 영상 생성

- 사진 → 짧은 클립 변환
- Live Photo → 영상 클립 추출
- 클립 자동 합성

#### C. BGM 추가

- 기본 음악 3~5개 제공
- 자동 적용 + 변경 가능

#### D. 미리보기

- 자동 재생
- 재생/일시정지 가능

#### E. 저장

- 카메라롤 저장
- 공유 가능

---

### 6.2 제외 기능

- 타임라인 편집
- 텍스트 삽입
- 스티커
- 필터
- SNS 직접 업로드
- 프로젝트 저장

### 9.1 입력

- Live Photo

---

### 9.2 영상 생성

#### 출력 규격

- 비율: 9:16
- 해상도: 1080x1920
- FPS: 30

#### 클립 길이

- 사진: 0.8~1.2초
- Live Photo: 1.5~2.5초

#### 규칙

- 선택 순서 유지
- 중앙 크롭
- 기본 컷 전환

---

### 9.3 BGM

### 9.4 저장
